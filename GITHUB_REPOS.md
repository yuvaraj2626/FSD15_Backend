## 📦 How to Clone and Run

### Backend Setup
```bash
# Clone the repository
git clone https://github.com/yuvaraj2626/FSD15_Backend.git
cd FSD15_Backend

# Install dependencies
npm install

# Create .env file with your MongoDB credentials
# MONGODB_URI=your_mongodb_connection_string
# JWT_SECRET=your_jwt_secret
# PORT=5000

# Seed the database (optional)
npm run seed

# Start the server
npm run server
```

### Frontend Setup
```bash
# Clone the repository
git clone https://github.com/yuvaraj2626/FSD15_Frontend.git
cd FSD15_Frontend

# Install dependencies
npm install

# Update API URL in src/services/api.js if needed
# Or use proxy in package.json

# Start the app
npm start
```
